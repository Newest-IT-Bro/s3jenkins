A practical IT project built to demonstrate infrastructure, deployment, security, automation, and operational troubleshooting skills.

## Overview

This repository contains the work for an end-to-end IT project covering areas such as:

- Cloud infrastructure
- Networking and security
- Application deployment
- Monitoring and logging
- Troubleshooting and validation
- Infrastructure as Code

The purpose of this project is to show hands-on ability in building, managing, and improving a working environment using real-world tools and practices.

:smile: Project Goals :smile:

The main goals of this project are:

- Build secure and reliable infrastructure
- Deploy an application in a structured environment
- Implement networking, access control, and DNS
- Use Infrastructure as Code for repeatability
- Validate functionality through testing and troubleshooting
- Document the setup clearly for future use

---

☂️ Key Areas Covered ☂️

- AWS infrastructure
- Terraform provisioning
- EC2 and application hosting
- Load Balancer configuration
- Route 53 DNS setup
- ACM certificate management
- WAF and security controls
- S3 logging
- Target group health checks
- HTTPS validation and troubleshooting
